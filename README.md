# Waitlist Management System

## Overview
This project manages participant waitlists for an event type using a structured, centralized system.

It allows staff to track waitlisted participants, manage status changes, and maintain a clear view of availability and movement over time.

## Features
- Centralized waitlist tracking
- Clear visibility into participant status and order
- Admin-facing interface for updating and managing entries

## Tech Stack
- PHP
- HTML
- JavaScript
- SQL / MySQL
- WordPress (context-dependent)

## Key Contributions
- Built system from the ground up to track waitlists for an event type
- Improved visibility into participant movement and status
- Reduced administrative overhead and confusion around ordering

## Notes
This tool was designed around real operational needs, with a focus on clarity and ease of use for non-technical staff.
